Project Implementation Guide

1. Project Overview

EcoKnowledge is an advanced Environmental Intelligence platform engineered to evaluate and predict the urban health of the Andhra Pradesh (AP) region. By dividing the state into precise 1km² geo-spatial grids, the platform utilizes Machine Learning to balance infrastructural growth with essential environmental preservation.

2. Core Technologies

The backend prediction engine is powered by a robust hybrid model combining Random Forest and XGBoost algorithms. These models were trained on historical ecological datasets to output an accurate Urban Health Index (UHI) score ranging from 1 (Critical) to 5 (Healthy).

To ensure transparent AI decision-making, we integrated SHAP (SHapley Additive exPlanations). This allows users to view Global Weights (overall feature importance) and Local Decision Paths (waterfall breakdowns showing exactly how specific parameters influenced a unique grid's score).

3. Key Parameters

Greenness (%) Percentage of area covered by active vegetation (NDVI). Crucial for carbon sequestration.
Urban Density Concentration of built infrastructural environments. High density negatively impacts health scores.
UHI Effect (°C) Urban Heat Island variation. Measures how much hotter the urban grid is compared to rural baselines.
LSTVI Dynamics Land Surface Temperature to Vegetation Index ratio. A key indicator of thermal stress.
PM 2.5 Levels (µg/m³) Fine particulate matter concentration. The primary metric for assessing immediate air quality threats.

4. How to Use the Platform

A. The Environmental Simulator

Navigate to the Simulation Dashboard to manually test hypotheses. Enter values for the 5 parameters and click "Analyze Environment". The ML API will process the data and return an Urban Health Score. The dashboard will instantly generate dynamic, actionable suggestions (e.g., policy changes or zoning restrictions) tailored to that specific score.

B. The AP EcoGrid Analysis (Map)

Navigate to the Interactive Map to explore geospatial data.

  • Colormap Tool: Use the top-right control panel to colorize the map based on specific variables (like PM 2.5 or Greenness).
  • Historical Data: Click on any highlighted 1km grid cell to open the Details Panel, featuring historical trend graphs (Summer vs Winter) for all metrics from 2000 to 2024.
  • Manual Geo-Analysis: Click anywhere outside the predefined grids to open the Urban Health Input Panel. This allows you to drop a custom pin, manually input surveyed data for that exact latitude/longitude, and generate a real-time health score widget for unmapped areas.
